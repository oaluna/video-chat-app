It is chat application, built with MERN stack.

Incoming features:-
1. File Attachments
2. Lazy Loading
3. Dark Mode
4. Mobile SDK

To install all dependencies
<code>npm install</code>

To start server
<code>npm run server</code>

To start client
<code>npm run client</code>

To run both concurrently
<code>npm run dev</code>
